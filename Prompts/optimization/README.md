# Prompt Optimization

> Meta-prompts and techniques for improving prompts.

## Contents

This folder contains prompt optimization techniques:

1. **Prompt Improver** — Analyze and enhance prompts
2. **Chain of Thought** — Step-by-step reasoning wrapper
3. **Few-Shot Learning** — Example-based learning template
4. **Self-Reflection** — Quality verification prompt

## Key Techniques

### 1. Clarity Improvement
- Remove ambiguity
- Add concrete details
- Use specific language

### 2. Structure Enhancement
- Organize logically
- Use clear sections
- Number steps

### 3. Constraint Addition
- Set output limits
- Specify format
- Define scope

### 4. Example Integration
- Show expected output
- Demonstrate patterns
- Provide templates

## Usage

Apply these meta-prompts to any prompt to improve its effectiveness.

See `COMBINED_PROMPTS.md` for full templates.
