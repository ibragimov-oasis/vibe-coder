# DevOps Role Agents

This directory contains AI agents specialized in **DevOps, infrastructure, and operations**.

## Agent Categories

### Infrastructure Engineers
- Cloud architecture (AWS, Azure, GCP)
- Infrastructure as Code (Terraform, Pulumi)
- Container orchestration (Kubernetes, Docker)
- Network configuration

### CI/CD Specialists
- Pipeline design and optimization
- GitHub Actions, GitLab CI, Jenkins
- Build automation
- Deployment strategies

### Site Reliability Engineers (SRE)
- Monitoring and alerting
- Incident response
- Performance optimization
- Capacity planning

### Security Operations
- Security scanning and hardening
- Compliance automation
- Secrets management
- Vulnerability assessment

## Usage

Agents in this directory are optimized for:
- Infrastructure automation
- Deployment orchestration
- System reliability
- Operational excellence

## Integration

These agents can be combined with:
- `security` - For security-first DevOps
- `architect` - For infrastructure design
- `coder` - For automation scripting
