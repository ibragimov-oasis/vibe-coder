# Scientist Role Agents

This directory contains AI agents specialized in **scientific research, analysis, and methodology**.

## Agent Categories

### Research Scientists
- Hypothesis generation and testing
- Experimental design
- Literature review and synthesis
- Data analysis and interpretation

### Data Scientists
- Statistical modeling
- Machine learning applications
- Data visualization
- Predictive analytics

### Domain Scientists
- Physics, chemistry, biology specialists
- Mathematical modeling
- Computational science
- Interdisciplinary research

## Usage

Agents in this directory are optimized for:
- Scientific method application
- Rigorous analysis
- Evidence-based reasoning
- Reproducible research

## Integration

These agents can be combined with:
- `researcher` - For literature review
- `analyst` - For data analysis
- `coder` - For scientific computing
