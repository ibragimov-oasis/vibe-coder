## What it does (plain language for vibe-coders)

The most comprehensive list of free software network services and web applications you can self-host. 220k+ stars — one of the most starred repos on GitHub. Covers 60+ categories: analytics, automation, CMS, e-commerce, DNS, document management, file transfer, games, groupware, learning, media streaming, monitoring, note-taking, password managers, and hundreds more. Also lists non-free alternatives separately. Maintained by the community with auto-link checking.
