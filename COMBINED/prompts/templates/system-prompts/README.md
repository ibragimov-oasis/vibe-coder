# System Prompts Reference

> Reference collection of system prompts from major AI tools.

## Contents

This folder contains references to system prompts from:

- **Anthropic** — Claude system prompts
- **Cursor** — AI IDE prompts
- **Devin** — Autonomous coding agent
- **Google** — Gemini and related tools
- **Lovable** — No-code app builder
- **Perplexity** — AI search engine
- **Replit** — Online IDE assistant
- **Windsurf** — Codeium's AI IDE
- **v0** — Vercel's UI generator

## Sources

- `system-prompts-and-models-of-ai-tools/` — Organized by tool
- `system_prompts_leaks/` — Historical prompt archives

## Purpose

Study these prompts to understand:

1. How major AI tools structure their instructions
2. Common patterns in system prompts
3. Safety and behavior guidelines
4. Tool usage patterns

## Key Patterns

- **Role Definition** — Clear AI identity
- **Capabilities** — What AI can/cannot do
- **Behavior Guidelines** — Response patterns
- **Safety Rules** — Refusal criteria
- **Output Format** — Structure requirements
