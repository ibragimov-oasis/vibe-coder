# Vibe-Coding Prompts

> Prompts specifically designed for vibe-coding workflows.

## Contents

This folder contains prompt templates for:

1. **Deep Research** — Comprehensive research prompt builder
2. **PRD Generation** — Product Requirements Document templates
3. **Technical Design** — Architecture and design templates
4. **Agent Notes** — Context handoff templates

## Source

These prompts are derived from `vibe-coding-prompt-template/` which provides a structured 4-part workflow:

1. Part 1 — Deep Research Prompt Builder
2. Part 2 — PRD MVP Generator
3. Part 3 — Technical Design MVP
4. Part 4 — Agent Notes

## Usage

Start with Part 1 to gather research, then proceed through Parts 2-4 to generate complete project documentation for your AI coding assistant.

See `COMBINED_PROMPTS.md` for the full prompt library.
